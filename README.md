# FutureCommander #

Projet for class:
   MTI835 - Développement d'applications graphiques

### Objectives ###

* Contrôle caméra RTS avec limites du terrain
* Charger et implémenter les bâtiments
* Rotation et positionnement de bâtiments
* Interface pour choisir les bâtiments
* Affichage de carte aérienne (bird’s eve view) 
* Sélection de bâtiments existants
* Création d’un terrain de base
* Création d’unités de combat
* Animation directionnelle de canon
* Projectile et effet d’impact
